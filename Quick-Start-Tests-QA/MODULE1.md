# Module 01: Overview
### Personal skills of a QA/Tester
1. Soft Skills
    - Motivation
    - Persistence
    - Curiosity
    - Studios
    - Perfectionism
    - Detail-oriented
    - Resilience
    - Organized
    - Self-managed
2. Hard Skills
    1. General
        1. S.O
            - Windows
            - Linux and Android
            - Mac and iOS
            - Mainframe
            - Command-line: DOS/Terminal
        2. Office
            - Text editor
            - Spreadsheet
            - Apresentation
        3. E-mail
            - Interpretation
            - New York Times Cover, don't write anything that couldn't be on the cover
        4. Languages
            - English
            - Mandarin
            - Others that you plan work with
        5. Internet
            - Saint Google, before call help, search it on google
        6. Security
            - Privacy, everything leaves a trace
            - Fragility, passwords and conections
            - Vulnerabilities, don't click or download files you don't recognize
    2. Business
        - Patterns
        - Templates
        - Certifications
        - Costumes
        - Rules
    3. Technology
        1. Programming
            - Logic
            - Language
        2. Telecomunication
            - Protocols
            - Networks
        3. Infrastructure
            - Virtualization, VMs and containers
            - Servers
            - Components
            - Equipaments
        4. Databases
            - Relational
            - Non-relational
    4. Test & QA
        - Plan
        - Analyze
        - Model
        - Implement
        - Execute, manual and automated
            - Communicate bugs, infos: how to reproduce; evidences: prints, videos, logs
            - Track bugs, confirmation and regression test
        - Techniques
        - Practices
        - Process
        - Strategies
        - Tools

### Interpersonal skills of a QA/Tester
1. Comunication
    - Listen
    - Speak
    - Posture
    - Read
    - Interpret
    - Write
    - Diagram
2. Negotiation
    - Deadlines
    - Resources
3. Empathy
    - Understand others desires, needs, pains, limitations, preferences, habits, costumes and beliefs
4. Team work
    1. Reunions
        - Goals: progress, doubts, decisions
        - Start: planning
        - Middle: dailys
        - End: review & retro
    2. Collaborate
        - Receive and give help
        - Receive and give feedbacks
        - Be transparent
        - Don't make decisions for the team
        - Don't be fool, learn to say no when necessary
