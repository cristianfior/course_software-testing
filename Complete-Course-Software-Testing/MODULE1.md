# Module 01: The Companies | The Professionals | The Opportunity
### What is Software Testing
>It's the verification made in a system or part of it to guarantee that a specific input always results in a expected output 
1. Tests doesn't fully verify the outputs of a system as the inputs are infinities.
2. Tests doesn't guarantee quality.
3. Tests cost about 20% to 30% of the total cost of a software elaboration.

### The companies current situation
- Tight deadlines;
- Intolerant clients/users;
- Necessity to build quality softwares;
- Less failure tolerance;

In summary: the companies need to build better softwares, faster and cheaper

- New technologies and systems more complex;
- There aren't specialists testers;
- Small investiment in tests;
- Software only tested by the devs;

In summary: poor software systems. "that's ok, we'll fix it in production"

### The professionals current situation
- There are few specialist testers;
- Aversion of the tester job;

**The profile of a tester should be:**
- Patient;
- Focused;
- Detail-oriented;
- Judicious;
- Motivaded;
- Insightful;
- Pessimist, destructive vision of a system;

### The Opportunity
Use the market shortage and specialize in software testing, you'll be a rare professional.
